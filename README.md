# 📸 Instagram Clone

A fully functional **Instagram Clone** built using **React Native** and **Expo**. This project recreates the core Instagram experience, including posts, stories, reels, profile management, messaging UI, and interactive features such as likes, comments, and bookmarks.

## ✨ Features

### 🏠 Home Feed

* Instagram-style home feed
* Like and unlike posts
* Save/bookmark posts
* Add and view comments
* Dynamic post rendering using `FlatList`

### 📖 Stories

* Horizontal stories section
* Story viewer with auto-progress
* Swipe down to close stories
* Animated progress indicators

### 🔍 Search

* Search users by username
* Grid layout for posts
* Real-time filtering

### 🎬 Reels

* Reels screen inspired by Instagram
* Vertical scrolling experience
* Video support using Expo Video

### 💬 Messaging

* Direct Messages screen
* Individual chat screen
* Send and receive text messages
* Auto-scroll to latest message
* Message bubble UI

### 👤 Profile

* User profile screen
* Edit Profile screen
* Profile statistics (Posts, Followers, Following)
* User posts grid

### 🎨 UI Features

* Bottom tab navigation
* Responsive layouts
* Reusable components
* Smooth animations
* Clean folder structure

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Vsr4661/Instagram-clone-.git
cd Instagram-clone-
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npx expo start
```

Press:

* `i` → Open iOS Simulator
* `a` → Open Android Emulator
* `w` → Open Web
